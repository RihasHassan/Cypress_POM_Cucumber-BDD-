Steps for execution :

It can be executed in two ways ,

1.TestRunner -> In terminal type 'npx cypress open' then it will open TestRunner .
2.Command Line->Type 'npx cypress run' or 'npm run execute' (Its a custome command created in package.json)


Multiple Cucumber HTML Reporter:

step1: In terminal type 'npm run execute'
step2: After execution is completed ,in termianl type 'node cucumber_html_report.js'
step3:It will create a html report under reports forlder ,open it with any browser.

(Generated html report is recommended to open in a live server option of VS code ,if its not installed in VS then please install it )
